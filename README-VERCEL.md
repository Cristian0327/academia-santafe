# Vercel Deploy - Academia Santafé

## Para subir a Vercel (GRATIS - Next.js funciona perfecto):

1. Ve a https://vercel.com
2. Click "Sign Up" → Usa tu cuenta de GitHub
3. Click "Add New..." → "Project"
4. Click "Import Git Repository"
5. Conecta tu GitHub y selecciona el repositorio
6. Click "Deploy"
7. LISTO - Tarda 2-3 minutos
